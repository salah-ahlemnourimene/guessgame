# Guessgame 

A simple web game where you guess a number between 0 and 10!

## How to play
- Enter a number between 0 and 10.
- Click on "Submit".
- If you guess the correct number, you win!
- Otherwise, try again.

## Technologies used
- HTML
- CSS
- JavaScript
